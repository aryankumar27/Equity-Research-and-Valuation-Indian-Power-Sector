# Equity-Research-and-Valuation-Indian-Power-Sector
Conducted FCFF and FCFE-based valuation of Tata Power, Adani Power, and Torrent Power using financial statement analysis, cost of capital estimation, and peer benchmarking. Compared intrinsic values with market prices to identify mispricing, finding Torrent Power overvalued, Tata Power near fair value, and Adani Power slightly undervalued.
